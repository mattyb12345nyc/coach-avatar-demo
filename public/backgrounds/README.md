Drop background images for the Active Session stage here. The keys
referenced in `src/config/backgrounds.ts` are:

- `coach-cherry-boutique.png` — **provided by Matt manually after scaffold**.
  Until this file lands, the Cherry Boutique preset falls back to a solid
  Coach black color.

  the real gold C-monogram asset whenever it's available.

Recommended dimensions: 1920x1080, JPEG/PNG/SVG all work.
